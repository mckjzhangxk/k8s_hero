[安装](https://developer.hashicorp.com/vagrant/downloads)

```
    vagrant init  //创建vagrant file
    vagrant up //启动
    vagrant ssh  
    vagrant reload //关闭，重启启动
    vagrant global-status //查看全局 启动的虚拟机
    vagrant box list
    vagrant box remove
    vagrant destory 

    configure("2") //版本
    comfig.vm.box //image 
    config.vm.network "public_network"
    //virtual box
    vb.memory 


```
